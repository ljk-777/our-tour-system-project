# /algo-bench — 算法性能对比基准测试

运行课程设计要求的全套算法对比，生成可用于验收材料的性能报告。

## 对比项目

### 1. TopK 对比（景点推荐）
- 方案 A：小顶堆 TopK — O(n log k)
- 方案 B：Quickselect — O(n) 平均
- 方案 C：全量快速排序 — O(n log n)
- 测试数据：n = 100 / 1000 / 10000 景点，k = 10

### 2. 最短路对比（路线规划）
- Dijkstra（距离权重）
- Dijkstra（时间权重，含拥挤度）
- 测试：随机生成 50/200/500 节点图，各跑 100 对随机起终点

### 3. 多点路径对比（TSP 近似）
- 最近邻贪心 + 2-opt
- Held-Karp DP（目标点 ≤ 12）
- 测试：目标点数 = 4 / 6 / 8

### 4. 文本检索对比（日记搜索）
- KMP 单模式匹配
- 倒排索引多文档检索
- 测试：1000 篇日记，关键词长度 3~15

### 5. 模糊查找对比（美食搜索）
- BK-Tree（编辑距离索引）
- Trie + 编辑距离阈值

## 输出格式
Markdown 表格，含耗时（ms）、内存（KB）、精度（%）三列。

运行方式：`node tests/unit/algorithms/bench.js`
